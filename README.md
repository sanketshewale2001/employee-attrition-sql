Employee Attrition Analysis using SQL

Dataset:
Kaggle Employee Dataset (4653 records)

Objective:
Analyze employee attrition and identify key drivers of employee exits.

Analysis Includes:
- Overall attrition rate
- Attrition by gender, city, age group
- Payment tier and experience impact
- Bench status influence

Skills Used:
SQL, Aggregations, CASE statements, Business Analysis

Outcome:
Identified critical factors contributing to employee attrition for HR decision-making.
